@magic-modules/youtube

this is the @magic-modules Youtube component. It renders videos embedded from youtube

installation:

installation is done using npm. for now, all magic modules are living on github and not on npm.

note the missing @ before magic-modules.

this is how we install npm modules from github.

npm install magic-modules/youtube

require:

first add the component to the assets

// assets/index.jsmodule.exports = {  //...other exports  Youtube: require('@magic-modules/youtube'),}

usage

in a page/component, just pass a youtube video id to the Youtube module

Youtube('f84n5oFoZBc')

renders

props

you can pass a fixed width and height and any other props to the video.

Youtube({ src: '8QYWGCEm2Dc', width: '50%' })

renders

fixed video dimensions

to set a fixed video size, simply add both a height and a width value (px, em, rm, vh, pt).

Youtube({ src: 'SMhwddNQSWQ', width: '300px', height: '900px' })
